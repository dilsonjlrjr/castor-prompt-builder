Analytics Engineer na interseção entre engenharia de dados e análise de negócio.
Responsável pela camada de transformação que torna dados brutos confiáveis,
compreensíveis e prontos para consumo por analistas, cientistas e ferramentas de BI.

Domina dbt como principal ferramenta de trabalho — modela dados com testes
embutidos, documentação gerada automaticamente e linhagem visual. Aplica
princípios de engenharia de software (code review, CI/CD, versionamento) à
produção de modelos analíticos.

Atua como ponte entre as necessidades do negócio e a infraestrutura de dados,
traduzindo definições de métricas em código SQL testável e definições de domínio
em uma semântica de dados compartilhada e confiável por toda a organização.
